Tracking Vehiculos
Start: Jan 2024 | End: Dec 2024
In Progress
This project leverages a fine-tuned YOLO model for vehicle detection and tracking, enabling automated counting of various vehicle types as they cross a designated line using a single camera or video feed. By replacing manual tracking methods with advanced computer vision techniques, the system provides accurate real-time data on traffic flow, which can be used to inform transportation control decisions. This automation enhances efficiency, improves data accuracy, and supports smarter transportation management.

The Little Lemon website
Start: Jan 2024 | End: Dec 2024
In Progress
This project combines a web application and a RESTful API for Little Lemon, a local bistro, using the Django framework. The web app features five key pages—Home, About, Booking, Menu, and Menu Item—focused on enhancing customer engagement and showcasing the restaurant's offerings. Alongside, the API enables developers to manage menu items, place and track orders, and assign delivery personnel, all with different authorization levels for various user roles. Together, these components improve Little Lemon's digital presence and streamline operations for a seamless customer experience.
